Agent-Orchestrierung · Claude Agent SDK

Wie meine Agenten arbeiten — und das Agent-SDK

Ich bin kein Entwickler, ein Dirigent. Ein Agent ist nicht magisch: Es ist ein Modell in einer Schleife. Es denkt, wählt ein Werkzeug, führt es aus, prüft das Ergebnis — und wiederholt das, bis das Ziel erreicht ist. Mein Job ist, diese Schleifen sauber aufzustellen und zusammenzuführen.

MCP · der Standard-Stecker

Agenten sprechen über MCP

Das Model Context Protocol ist der einheitliche Stecker zwischen Agent und Werkzeug. Derselbe Agent erreicht so Notion, Linear, Supabase, n8n und den Browser — ohne Spezial-Code je Tool.
